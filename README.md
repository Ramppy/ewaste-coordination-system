# Web-Based E-Waste Coordination and Recycling System (EWCS)

A robust full-lifecycle management platform featuring a functional web architecture designed to automate local e-waste recycling workflows, streamline coordination frameworks, and optimize material tracking.

##  Key Features
- **Coordination Frameworks:** Automated request workflows connecting e-waste holders with recycling centers.
- **Relational Database Architecture:** Optimized schema design mapping localized lifecycle stages and disposal records.
- **Secure Backend API:** Built using scalable back-end logic models to handle system state constraints and traffic tracking.

##  Tech Stack
- **Backend:** Python, Flask API
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript
